# Predicting-Online-Advertisement-Click-Behavior-Using-Logistic-Regression


This project analyzes online advertising data to understand the factors that influence whether users click on digital advertisements. By applying data science and machine learning techniques, the study examines user characteristics such as browsing behavior, internet usage, age, and income to identify patterns associated with advertisement engagement. The goal is to develop a predictive model that helps businesses better understand user behavior and improve advertising strategies.

# Research Question:
Can user behavioral and demographic characteristics be used to accurately predict whether a user will click on an online advertisement?

# Methods Employed:

The project follows the standard data science workflow, including:

1. Data exploration and cleaning

2. Exploratory data analysis and visualization

3. Feature selection and encoding of categorical variables

4. Logistic regression modeling

5. Model evaluation using performance metrics such as:

6. Confusion Matrix

7. Accuracy

8. Precision

9. Recall

10. F1-score

11. Data  visualisation


# Tools:

1. Python

2. Pandas (data manipulation)

3. NumPy (numerical computation)

4. Matplotlib & Seaborn (data visualization)

5. Scikit-learn (machine learning and model evaluation)


# Key Findings

The model showcases strong predictive performance with 95% accuracy. The confusion matrix shows that the model correctly identified most users who clicked on advertisements as well as those who did not, while Precision, recall, and F1-scores for both classes were approximately 95%, demonstrating that the model performs consistently and maintains a good balance between correctly identifying ad clickers and non-clickers. Overall, the results suggest that user behavior patterns can be effectively used to predict advertisement engagement.
